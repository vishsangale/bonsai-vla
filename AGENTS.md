# Agent Instructions

## Environment Management
- **ALWAYS** use the virtual environment located at `.venv/` for all Python-related tasks.
- When running commands, prefer using the absolute or relative path to the venv executable: `./.venv/bin/python3`.
- Do not install packages globally; use `./.venv/bin/pip`.

## Coding Style
- Follow the "From Scratch" philosophy: use minimal high-level abstractions from libraries like `diffusers` or `timm` unless explicitly requested.
- Prioritize readability and education.
